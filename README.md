[FFC_ToHDR](https://github.com/OpenPCB/FFC_ToHDR)
========
* Author  : [Hom](https://github.com/Hom-Wang)
* Version : v2.0
* Update  : 2016/01/12

Description
========
FFC_ToHDR 是一個 FFCSPI 的轉接板，將 FFC16 轉成 Header10。
